Aula 1 - A diferenca entre HTML, CSS e JavaScript

HTML é linguagem de conteúdo
Ex: textos / imagens / videos / tabelas / listas

CSS é design
Ex: cores / sombras / tamanhos / posicionamento

JavaScript (JS) é linguagem de interações
Ex: menus / animações / popups

Estrutura basica HTML
